### 方案 3: Tushare（基础版免费）⭐⭐⭐

**网址**: https://tushare.pro

**免费额度**:
- 注册送 100 积分
- 基础数据免费（日线、财务等）
- 资金流向需要 120 积分（需要签到或捐赠）

**优势**:
- ✅ 数据规范
- ✅ 接口稳定
- ✅ 文档完善

**劣势**:
- ⚠️ 需要注册
- ⚠️ 高级数据需要积分

**使用示例**:
```python
import tushare as ts

# 注册后获取 token
ts.set_token('your_token')
pro = ts.pro_api()

# 获取日线数据（免费）
df = pro.daily(ts_code='000001.SZ', start_date='20260101')

# 资金流向（需要积分）
df = pro.moneyflow(ts_code='000001.SZ', trade_date='20260316')
```
