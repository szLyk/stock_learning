# 小市值股票特征分析方案

## 一、分析目标

1. 建立小市值股票特征档案
2. 发现主升浪启动的规律性特征
3. 提高主升浪判断准确率

---

## 二、数据准备

### 2.1 筛选条件

**市值筛选**：流通市值 < 100亿

**行业筛选**：
- 科技股：计算机、通信、电子、软件、半导体
- 资源股：有色金属、煤炭、石油、矿业
- 可扩展：新能源、医药、消费

### 2.2 数据来源

| 数据表 | 用途 |
|--------|------|
| stock_profit_data | 流通股本、总股本 |
| stock_history_date_price | 日线数据、换手率、涨停 |
| stock_industry | 行业分类 |
| date_stock_moving_average_table | 均线数据 |
| date_stock_macd | MACD数据 |

---

## 三、特征分析维度

### 3.1 基础特征

| 特征 | 计算方法 | 用途 |
|------|----------|------|
| 流通市值 | 流通股本 × 最新价格 | 市值分类 |
| 年化波动率 | 收益率标准差 × √250 | 波动特征 |
| 平均换手率 | 60日平均换手率 | 活跃度 |
| 涨停频率 | 年化涨停次数 | 强势特征 |

### 3.2 主升浪特征

| 特征 | 计算方法 | 用途 |
|------|----------|------|
| 主升浪成功率 | 历史主升浪次数/涨停次数 | 启动有效性 |
| 主升浪平均涨幅 | 历史主升浪平均涨幅 | 收益预期 |
| 主升浪持续时间 | 平均持续天数 | 持仓周期 |
| 换手率放大倍数 | 主升浪期/蓄势期换手率 | 启动信号 |

### 3.3 行业特征

| 特征 | 分析内容 |
|------|----------|
| 行业热度 | 板块涨跌幅、资金流入 |
| 龙头效应 | 行业龙头走势 |
| 板块轮动 | 与其他板块的关联性 |

---

## 四、分析流程

```
步骤1: 筛选小市值股票
   ↓
步骤2: 按行业分类（科技、资源等）
   ↓
步骤3: 提取历史主升浪数据
   ↓
步骤4: 计算各维度特征
   ↓
步骤5: 建立特征档案
   ↓
步骤6: 分析不同行业特征差异
```

## 七、需要确认

1. **行业范围**：先分析科技股、资源股，还是扩大范围？
2. **市值阈值**：小市值定义为<100亿，还是<50亿？
3. **样本量**：分析100只、300只还是全部小市值股票？
4. **分析周期**：分析近1年还是3年的数据？
