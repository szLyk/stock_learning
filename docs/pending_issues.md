# 待改进问题清单

## 一、数据问题

### 1. 市值数据
- **问题**：用当前市值分类历史股票
- **影响**：市值会变化，小盘股一年内可能变成中盘
- **优先级**：高
- **建议**：采集历史市值数据（每日流通市值）

### 2. 未来函数
- **问题**：原验证用信号日收盘价作为买入价
- **影响**：胜率高估2-3%
- **优先级**：高
- **建议**：改用次日开盘价

### 3. 市值估算方式
- **问题**：通过换手率反推，有误差
- **影响**：市值分类可能不准确
- **优先级**：中
- **建议**：接入Tushare获取真实市值

## 二、验证逻辑问题

### 1. 未考虑交易成本
- **问题**：未扣除手续费、印花税、滑点
- **影响**：实际收益更低
- **优先级**：中
- **建议**：回测时扣除千分之三成本

### 2. 停牌处理不完善
- **问题**：停牌股票后续数据为空，直接跳过
- **影响**：可能有偏差
- **优先级**：低
- **建议**：标记停牌情况，单独统计

### 3. 涨跌停限制
- **问题**：未考虑涨跌停无法买入
- **影响**：极端情况下无法操作
- **优先级**：低
- **建议**：过滤涨停股
