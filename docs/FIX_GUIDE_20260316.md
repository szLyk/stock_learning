## 🚨 常见问题

### Q1: 执行 SQL 脚本报错 "Access denied"
**解决**：确保使用 root 用户或有 ALTER 权限的用户执行

### Q2: 仍然报 "Unknown column" 错误
**解决**：
1. 检查 SQL 脚本是否执行成功
2. 手动验证表结构：`DESC stock_balance_data;`
3. 确认连接的数据库是 `stock`

### Q3: 数据采集仍然失败
**解决**：
1. 查看日志：`tail -f logs/baostock_extension.log`
2. 检查 Redis 缓存是否已清理
3. 运行单只股票测试：`python tests/test_single_stock.py --stock sh.600000`
