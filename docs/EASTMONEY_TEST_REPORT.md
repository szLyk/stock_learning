## 🐛 发现的问题

### 1. 部分数据缺失

**问题:** 股东人数、概念板块、分析师评级返回空数据

**影响:** 低

**解决:** 
- 改用其他股票测试
- 使用替代数据源（Tushare）
- 优化 API 参数

### 2. 收盘价数据异常

**问题:** 资金流向接口返回的 `close_price` 为负数

**影响:** 中

**解决:** 从 `stock_history_date_price` 表获取正确收盘价

### 3. 证券 ID 转换

**问题:** 需要手动转换股票代码格式

**影响:** 低

**解决:** 已内置 `_get_secid()` 方法自动转换
