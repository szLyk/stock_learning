**日期**: 2026-03-17  
**测试股票**: 浦发银行 (600000)

---

## 📊 当前状态

### 可用的 AKShare 接口

1. **同花顺概念板块列表** ✅
   ```python
   df = ak.stock_board_concept_name_ths()
   # 返回所有概念板块：name, code
   ```

2. **东方财富概念板块列表** ⚠️（网络不稳定）
   ```python
   df = ak.stock_board_concept_name_em()
   ```

### 不可用的接口

1. **获取个股所属概念** ❌
   - `ak.stock_individual_info()` - 接口不存在
   - `ak.stock_board_concept_cons_ths()` - 接口不存在

2. **获取概念成分股** ❌
   - 网络不稳定，经常超时

---

## 🎯 解决方案

### 方案 1：使用缓存的概念 - 股票映射表（推荐）

**思路：**
1. 手动维护一个概念 - 股票映射表
2. 或定期从其他数据源获取并缓存
3. 采集时直接查缓存表

**优点：**
- 不依赖 AKShare 不稳定接口
- 速度快
- 稳定可靠

**缺点：**
- 需要定期更新缓存

---

### 方案 2：使用 Tushare 数据源

**Tushare 接口：**
```python
import tushare as ts

ts.set_token('your_token')
pro = ts.pro_api()

# 获取个股所属概念
df = pro.concept_detail(ts_code='600000.SH')
```

**优点：**
- 接口稳定
- 数据准确
- 有官方维护

**缺点：**
- 需要 token
- 有积分限制

---

### 方案 3：暂时跳过概念板块采集

**建议：**
- 先使用分析师评级、资金流向（部分可用）、股东人数接口
- 概念板块数据暂时跳过
- 等待 AKShare 接口稳定或切换到 Tushare
